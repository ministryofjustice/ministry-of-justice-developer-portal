## Slack Channel

The quickest way to get help is to post a question in the
[#ask-modernisation-platform](https://mojdt.slack.com/archives/C01A7QK5VM1) Slack channel.

## Create a GitHub Issue

You can also [raise a GitHub Issue](https://github.com/ministryofjustice/modernisation-platform/issues/new/choose) on our GitHub repository if you wish to raise a bug, make a feature request or request out-of-hours support.
